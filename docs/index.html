NordAPI: Production-ready .NET SDKs for Swedish payment integrations

NordAPI provides a secure .NET SDK that simplifies integration with Swish and BankID services. Our Swish API for .NET handles payment requests, webhook verification, and mTLS authentication out of the box. For BankID integration for .NET, we offer the same level of security and developer experience. Built specifically for .NET 8 and later, the SDK includes HMAC signing, nonce replay protection, and structured logging. Whether you're building e-commerce platforms, financial applications, or government services, our secure .NET SDK reduces integration time from weeks to hours while maintaining production-grade security standards.

Quickstart (5 minutes)

Install package

dotnet add package NordAPI.Swish

NuGet package contains client, models and helpers for Swish.

Basic payment (sandbox)

var swish = new SwishClient(cfg => cfg.UseDevDefaults());
await swish.PayAsync(new SwishPaymentRequest {
    Amount = 100m, Currency = "SEK", PayerAlias = "46712345678",
    Message = "Order #123"
});

Configure certs + mTLS for prod. See checklist below.

Webhook (verified)

app.MapPost("/swish/webhook", async (HttpRequest req) => {
    var evt = await SwishWebhook.ParseAsync(req);
    if (!evt.IsValid) return Results.Unauthorized();
    // handle payment success/failed here
    return Results.Ok();
});
